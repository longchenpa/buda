BDRC/BUDA Publishing System
===========================

```
$ escript pub
PUB Tibetan Digital Library Publishing System
Copyright (c) Longchen Nyingthig Ukraine

Usage:
       pub i          -- print meta index
       pub s          -- meta index with sizes for available volumes
       pub d          -- show only available volumes (filesystem tree output)
       pub u          -- show only unavailable volumes that should be downloaded
       pub h          -- print HTML index
       pub t <file>   -- import TBRC index from file by authors
       pub tp <file>  -- import plain TBRC index from file
       pub f <text>   -- search in everything
       pub fd <text>  -- search in available
       pub ft <f> <s> -- search in TBRC index
       pub w <file>   -- EWTS wylie file transcoding
       pub dump       -- dump meta index in Erlang format
       pub repl       -- REPL
       pub tex <file> -- publish TeX file
       pub tex        -- publish folder with TeX, PDF, TibetDoc, TXT files
```
